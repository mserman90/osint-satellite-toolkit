Quick Start: Nighttime Lights Template

Copy this snippet into your Google Earth Engine console to start analyzing VIIRS data:

var VIIRS = ee.ImageCollection("NOAA/VIIRS/DNB/MONTHLY_V1/VCMCFG").select('avg_rad');
var startDate = '2023-01-01';
var endDate = '2024-01-01';
var filtered = VIIRS.filterDate(startDate, endDate);
Map.addLayer(filtered.median(), {min: 0, max: 20}, 'Median Radiance');
